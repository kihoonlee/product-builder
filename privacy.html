AI 수학 문제 풀이 서비스의 개인정보 처리 방식을 안내합니다

시행일: 2026년 3월 1일  |  최종 수정일: 2026년 3월 1일

AI 수학 문제 풀이(이하 "서비스")는 이용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」 및 관련 법령을 준수합니다. 본 방침은 서비스가 어떤 정보를 수집하고 어떻게 사용하는지 설명합니다.

1. 수집하는 정보

본 서비스는 최소한의 정보만 처리하며 별도의 회원가입이나 개인정보 입력을 요구하지 않습니다.

  • •Gemini API 키: 이용자가 직접 입력한 API 키는 해당 기기의 브라우저 로컬 스토리지에만 저장됩니다. 운영자 서버로 전송되거나 저장되지 않습니다.
  • •업로드 이미지: 수학 문제 사진은 Gemini AI 풀이 요청 시 Google Gemini API에 전송되며, 서비스 운영자 서버에는 저장되지 않습니다.
  • •서비스 이용 정보(자동 수집): Google AdSense 등 제3자 도구를 통해 IP 주소, 브라우저 유형, 방문 페이지 등의 통계 정보가 자동으로 수집될 수 있습니다.

2. 정보의 이용 목적

  • •수학 문제 AI 풀이 서비스 제공 (Gemini API 연동)
  • •서비스 품질 개선 및 통계 분석
  • •광고 서비스 제공 (Google AdSense)

3. Google AdSense 광고

본 서비스는 Google LLC가 제공하는 광고 서비스인 Google AdSense를 이용합니다. Google AdSense는 이용자의 관심사와 관련된 광고를 제공하기 위해 쿠키 및 유사 추적 기술을 사용합니다.

  • •이용자는 Google 광고 설정 페이지에서 맞춤 광고를 비활성화할 수 있습니다.
  • •또는 NAI 옵트아웃 페이지에서 제3자 광고 쿠키를 거부할 수 있습니다.
  • •Google의 데이터 처리 방식은 Google 개인정보처리방침을 참고하세요.

4. Google Gemini API

업로드된 수학 문제 이미지는 풀이 요청 시 이용자의 브라우저에서 직접 Google Gemini API로 전송됩니다. 운영자는 이 이미지를 저장하거나 별도로 처리하지 않습니다.

  • •Gemini API를 통해 전송된 데이터는 Google AI 서비스 약관의 적용을 받습니다.
  • •이용자의 Gemini API 키는 운영자가 수집하거나 저장하지 않습니다.

5. 쿠키(Cookie) 사용

본 서비스는 직접 쿠키를 발행하지 않습니다. 그러나 아래 제3자 서비스가 쿠키를 사용할 수 있습니다.

  • •Google AdSense: 광고 관련 쿠키 (이용 기반 광고 제공 목적)
  • •Google Analytics: 방문자 통계 분석 목적의 쿠키

6. 개인정보의 보유 및 파기

  • •API 키: 이용자가 브라우저 로컬 스토리지를 직접 삭제하거나 "API 키 변경" 기능을 통해 언제든지 삭제할 수 있습니다.
  • •이미지: 업로드된 이미지는 서버에 저장되지 않으므로 세션 종료 시 자동으로 소멸됩니다.

7. 아동의 개인정보 보호

본 서비스는 만 14세 미만 아동을 대상으로 개인정보를 수집하지 않습니다. 만 14세 미만 이용자는 반드시 법정 대리인의 동의를 받은 후 이용해야 합니다.
